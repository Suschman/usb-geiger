USB Geiger counter
2013 Michał Słomkowski

Licencja: GNU GPL wersja 3.0.

Program ten udostępnia wszystkie funkcje licznika Geigera z poziomu linii komend. Listę można sprawdzić komendą:

./geiger-controller --help

Program został napisany w języku C++ z wykorzystaniem biblioteki libusbx, która powinna pozostać kompatybilna z libusb-1.0. Kompilowano i testowano w środowisku Linux, lecz nie powinno być problemów z uruchomieniem pod Windows. Program nie nie będzie rozwijany, funkcje takie jak wysyłanie na cosm.com, do bazy SQL czy GUI udostępnia skrypt w języku Python.
